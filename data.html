Course Description

This course provides a comprehensive journey into the world of Data Science and Artificial Intelligence. Students will learn to extract actionable insights from structured and unstructured data, build predictive models using supervised and unsupervised learning, and explore the frontier of Deep Learning. Emphasis is placed on the entire data lifecycle—from ingestion and cleaning to deployment and model monitoring.

Learning Objectives

  • Apply statistical methods to analyze and interpret complex datasets.
  • Master Python libraries for data science (Pandas, NumPy, Scikit-Learn).
  • Design and implement Machine Learning pipelines for classification and regression.
  • Understand the mathematical foundations of Gradient Descent and Backpropagation.
  • Build and train Convolutional Neural Networks (CNNs) for image recognition.
  • Apply Natural Language Processing (NLP) techniques to text data.
  • Execute Exploratory Data Analysis (EDA) and create impactful visualizations.
  • Implement Big Data solutions using Spark and distributed computing.
  • Address Algorithmic Bias and ensure ethical AI deployment.
  • Deploy models as APIs using Flask, FastAPI, or Docker.
