Course Description

Software Engineering is a foundational course that focuses on the systematic approach to the development, operation, maintenance, and management of software systems. The course introduces students to software development life cycle (SDLC) models, requirements engineering, software design, implementation, testing, maintenance, and project management. Emphasis is placed on producing high-quality, reliable, and maintainable software using engineering principles, tools, and best practices.

Learning Objectives

  • Understand the principles and concepts of software engineering.
  • Apply software development life cycle (SDLC) models in real-world scenarios.
  • Analyze and document software requirements effectively.
  • Design software systems using appropriate architectural and design techniques.
  • Implement software solutions using structured and object-oriented approaches.
  • Perform software testing, debugging, and quality assurance.
  • Understand software maintenance and evolution processes.
  • Apply basic project management and teamwork skills in software development.
  • Use modern software engineering tools and methodologies.
  • Develop ethical and professional responsibility in software practice.
